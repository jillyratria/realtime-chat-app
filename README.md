# Super Simple Realtime Chat App

## Getting Started

First, run the client development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Then, run the server:

```bash
npm run server
# or
yarn server
```
